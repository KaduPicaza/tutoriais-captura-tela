Métodos de Captura de Tela no Windows

Usando Teclas de Atalho

  • Print Screen (PrtScn): Captura toda a tela e copia para a área de transferência.
  • Alt + Print Screen: Captura apenas a janela ativa.
  • Windows + Print Screen: Captura toda a tela e salva automaticamente na pasta Imagens > Capturas de Tela.
  • Windows + Shift + S: Abre a ferramenta Recorte para selecionar uma área específica.

Para mais detalhes, consulte o tutorial completo do Canaltech.

Ferramentas Nativas do Windows

Ferramenta de Captura

O Windows possui ferramentas nativas para capturas de tela mais avançadas:

  • Ferramenta de Captura: Disponível em versões mais antigas do Windows.
  • Ferramenta Recorte e Esboço: Versão mais moderna disponível no Windows 10 e 11.

Para acessar, pesquise por "Captura" ou "Recorte" no menu Iniciar.
